# Agent Browser / Browser Harness Intake

Agent-browser and Browser Harness are high-priority future computer-use lanes.

## Direction

- Use browser tools for compliant QA and local app verification.
- Keep UI-TARS observation-only until a later audited control milestone.
- Require human approval for any action with real-world effects.
- Prefer local preview/testing over live accounts.

## Blocked

- bot-detection bypass
- captcha bypass
- fake engagement
- spam
- unauthorized scraping
- credential theft
- unauthorized desktop control

## Current Status

No agent-browser or Browser Harness runtime wiring is enabled in this milestone.
