# Blocked Unsafe Automation

The following remain blocked:

- bot detection bypass
- captcha bypass
- cloak browser bypass
- fake engagement
- spam
- account abuse
- credential theft
- unauthorized scraping
- unauthorized desktop control
- autonomous money, trading, or legal actions

Ghoti may document these risks, but must not implement bypasses or unsafe
automation. Any future browser/computer-use lane must be compliant, local-first
where possible, and human approval-gated.
