# Ghoti Documenso vs Overleaf Project Docs Lane

Documenso and Overleaf solve different document workflow problems.

## Classification

Documenso: signing, approval workflow, contracts, agreements, and document completion trails.

Overleaf: collaborative LaTeX editing, technical reports, university papers, thesis-style documents, and reproducible source-controlled writing.

## Self-Hosting And Accounts

Both projects have open-source repos, but N+6.19A does not host, deploy, configure accounts, or connect email. Ghoti records them as future lanes only.

## Ghoti Recommendation

1. Use an Overleaf/LaTeX export helper first for university and technical reports.
2. Use Documenso later for signed agreements, business contracts, approvals, and evidence trails.
3. Keep both behind local-first planning until a separate setup milestone handles accounts, hosting, authentication, and privacy.

## Why Both Matter

Overleaf helps create the report. Documenso helps get the finished document approved or signed. Ghoti should not treat them as substitutes.

- no Overleaf hosting
- no account setup
