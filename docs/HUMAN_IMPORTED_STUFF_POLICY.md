# Human Imported Stuff Policy

Raw user-provided files are private intake by default.

## Detected Intake Folder

- `C:\Users\ai_sandbox\Documents\AI_Managed_Only\Human Placed Stuff`

This folder is ignored and must not be committed directly.

## Allowed Curated Copies

Only normal image formats may be copied into `docs/assets/github/` after review:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.gif`
- `.svg`

The N+5.1A curated images are diagram-only assets with no visible secrets,
emails, private accounts, account screenshot content, school records, CV data,
private data, or personal documents.

## Exclusion Rules

Exclude raw or copied files if they appear to contain:

- secrets, tokens, passwords, private keys, or credentials
- emails or private account screens
- school/private docs
- CV/private docs
- account screenshots
- private documents or personal records
- copyrighted external media that is not clearly owned or safe to publish

When unsure, exclude the file and document the reason.
