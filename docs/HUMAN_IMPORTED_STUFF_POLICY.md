# Human Imported / Placed Stuff Policy

Human Imported Stuff and Human Placed Stuff folders are treated as raw private
intake.

## Rules

- Do not delete raw user files.
- Do not commit raw user-imported contents by default.
- Keep raw folders ignored.
- Copy only safe curated image assets into `docs/assets/github/`.
- Exclude anything that appears to contain secrets, API keys, tokens, emails,
  private accounts, private school/CV details, personal documents, or account
  dashboards.
- If unsure, exclude and document.

## Allowed Curated Formats

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.gif`
- `.svg`
