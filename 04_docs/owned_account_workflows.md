# Owned Account Workflows

## Allowed

- inbox triage summaries
- draft generation
- LinkedIn update planning
- CV update scaffolding
- GitHub profile and project organization planning
- documentation and report generation
- Notion update planning
- handoff summary creation

## Approval-Required

- editing important profile text
- preparing outbound outreach drafts for use
- publishing or posting
- sending mail or messages
- changing important docs or configs
- connecting live integrations

## Not Allowed

- spam outreach
- fake identities
- bulk cold-message automation
- mass fake engagement
- deceptive platform behavior
