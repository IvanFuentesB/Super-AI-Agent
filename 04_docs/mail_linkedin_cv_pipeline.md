# Mail LinkedIn CV Pipeline

## Pipeline

1. raw input
2. context extraction
3. draft pack
4. approval gate
5. final asset
6. optional publish or send step
7. handoff and memory update

## Examples

- inbox triage: mailbox context -> triage pack -> approved follow-up drafts
- LinkedIn refresh: profile context -> update pack -> approved profile edits or posts
- CV update: role target -> CV pack -> approved resume revision
- outreach draft: recipient context -> draft pack -> approved send later
