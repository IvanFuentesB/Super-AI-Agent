# Internship Ops Plan

## Goals

- evaluate internship and junior-role fit before applying
- prepare CV, LinkedIn, GitHub, and portfolio assets with less manual thrash
- generate case-study and evidence packs that support credible applications

## Useful Assets

- CV update packs
- LinkedIn update packs
- GitHub profile and repo organization plans
- portfolio and case-study scaffolds
- internship application packs

## Role Of Career-Ops

`career-ops` is reference material for scoring, tracking, and tailored asset generation. It is not the core runtime and should not be copied wholesale.

## Role Of Approval And Human Review

- humans should review role fit, messaging, and final assets
- submissions, messages, and profile edits should stay approval-gated
- the system should help prepare, not silently act

## Out Of Scope

- spray-and-pray applications
- fake identities
- deceptive outreach
- automated submissions without review
