# Automation Boundaries

## Allowed

- reading local files
- generating docs and reports
- owned-account workflow planning
- report scaffolding
- project and task organization
- legitimate content ops planning
- summarizing and structuring context

## Approval-Required

- writing files
- multi-file changes
- commits and pushes
- package installs
- connecting integrations
- external publishing actions
- changing important configs

## Forbidden

- fake account creation
- bulk email account creation
- platform abuse
- spam networks
- CAPTCHA bypass
- deceptive identity automation
- mass abusive account management
