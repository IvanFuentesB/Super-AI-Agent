# Showcase Plan

## Purpose

Define the first recruiter-friendly showcase for this repo without overstating what is live.

## Demo Now

- GitHub controlled workflow with readable status, branch, approval, and remote-capability checks
- Internship application pack generation from the runtime CLI
- Structured personal-ops and career-ops layer with durable context and reviewable outputs

## Wait For Later

- browser or app execution
- live Notion writes
- live mail sending or mailbox mutation
- broad multi-model orchestration claims

## Strongest Demo Angles

- GitHub controlled workflow: read-only inspection, approval-gated write commands, and non-mutating smoke readiness
- Internship application pack generation: visible, practical output tied to real job-search use
- Structured career-ops and personal-ops layer: reusable packs, intake research, and durable handoff files

## What Recruiters Should See

- the system is execution-first, not just a notes repo
- the runtime produces concrete artifacts
- risky actions are explicit and approval-gated
- third-party repos are handled intentionally as reference material, not dumped into core

## Not Ready Yet

- browser executor
- live external provider adapters
- live Notion integration
- polished remote GitHub smoke demonstrations by default
