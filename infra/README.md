# Ghoti Infrastructure

This folder is a placeholder-only infrastructure planning lane. It contains no
credentials and does not provision resources.

Default direction:

- local-first development
- Vercel for frontend/deploy
- Supabase for Postgres/auth/storage where practical
- Stripe, Resend, PostHog, Sentry, Upstash, and Pinecone only when a project
  actually needs them
- no paid VPS by default
- no real credentials committed
