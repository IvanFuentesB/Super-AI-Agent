# Specialized Agent Template

Copy this folder to create a new specialized automation or agent.

Suggested naming:

- `email_triage_agent`
- `repo_eval_agent`
- `benchmark_runner_agent`
- `notion_sync_agent`

Minimum docs to add when instantiated:

- `docs/overview.md`
- `memory/current-context.md`
- `tasks/backlog.md`
