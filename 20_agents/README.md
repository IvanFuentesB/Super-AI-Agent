# Agents Area

This directory is for specialized agents and automations.

Structure:

- `_templates/` - reusable skeletons
- `_shared/` - shared prompts, conventions, and supporting docs
- `future_agents/` - actual future agent folders

Guiding rule:

- shared sandbox truth belongs in `14_context`
- specialized behavior, task state, and short-lived working memory belong in the specific agent folder
