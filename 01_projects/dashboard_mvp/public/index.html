  • Operator status API — real
  • Approval queue + hardening — ~95% real
  • Ollama probe — real
  • Frame observer routes — real scaffolding, awaiting vision model
  • Voice — scaffold only
  • YouTube follower — scaffold only
  • Real voice / browser / desktop actions — not implemented
