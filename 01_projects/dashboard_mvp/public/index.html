About Ghoti

Project summary

Ghoti is a local-first supervised operator framework. Every action is approval-gated, manually executed, and fully traced. Nothing runs autonomously. The operator decides what happens at every step.

  • Active Mode capture / gallery / cleanup — ~90% real
  • Browser overlay (polling status) — ~85% real, not native
  • Operator status API — real
  • Approval queue + hardening — ~95% real
  • Ollama probe — real
  • Frame observer routes — real scaffolding, awaiting vision model
  • Voice — scaffold only
  • YouTube follower — scaffold only
  • Real voice / browser / desktop actions — not implemented
