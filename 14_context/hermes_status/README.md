# Hermes Status Packet (N+6.25A)

A reliable, read-only coordinator packet Hermes reads **before answering**, so it stops
giving shallow or incorrect summaries.

## Files here

- `hermes_status_packet_schema.json` - the JSON packet shape + safe posture.
- `HERMES_STATUS_PACKET.example.md` - a static, illustrative example of the rendered packet.
- `HERMES_STATUS_PACKET_LAST_RUN.md` - generated by `--write`; **not committed** (run output).

The generator is `03_scripts/hermes_status/ghoti_hermes_status_packet.py`.

## What it reads (committed sources only)

- latest main commit (read-only git metadata; `origin/main` preferred)
- Agent Arena trace status (if present)
- Memory Vault index (if present)
- Tool Intake summary (if present)
- latest Claude report and latest Codex report (milestone-sorted)
- feature-flags summary (the `*.example` flags file only - no secrets)

## How to use it

```
# Safety self-check (no write):
python 03_scripts/hermes_status/ghoti_hermes_status_packet.py --check --json

# Print the JSON packet (machine-readable):
python 03_scripts/hermes_status/ghoti_hermes_status_packet.py --json

# Print the Markdown packet:
python 03_scripts/hermes_status/ghoti_hermes_status_packet.py --md

# Write the Markdown packet for Hermes to read:
python 03_scripts/hermes_status/ghoti_hermes_status_packet.py --write \
  --output 14_context/hermes_status/HERMES_STATUS_PACKET_LAST_RUN.md --json
```

Then **paste the Markdown packet into Hermes** at the start of a coordination session (see
`docs/GHOTI_N6_25A_HERMES_MEMORY_STATUS_UPGRADE.md`). Hermes treats it as ground truth for
status and must not claim anything the packet marks blocked.

## Safety

Read-only. The only command run is read-only git (commit metadata). No agent launch, no
MCP, no browser/computer-use, no auto-submit, no secrets. It writes nothing unless
`--write` is passed, and then only to a path **inside the repo**. `ECC = Everything Claude
Code` (not elliptic curve cryptography). Hermes is **coordinator / status / memory only**
today; automatic coordination comes only after the controlled launcher and approval gates
are built and audited.
