# Codex N+3.52 - Bridge To 90 Percent Gap Analysis

## Percentage Truth

Current main after N+3.49A remains around `73-75%` toward the desired local-first supervised agent system.

If N+3.50A were pushed and merged as-is, the project would likely reach about `78-80%`, not 90%. Important surfaces exist, but the strongest capabilities are still unproven or manual.

Expected after a strong N+3.51A hardening implementation: `84-87%`.

## What N+3.50A Adds

- Local orchestrator dashboard visibility.
- Ruflo install gate shape.
- Gemma compression worker shape.
- Prompt bus context-pack generation.
- Obsidian helper improvements.
- More local worker routing.

## What Is Still Missing

1. N+3.50A is not pushed remotely.
2. Ruflo dependencies are not installed.
3. npm is not found in PATH.
4. Ruflo is not wired as an orchestrator.
5. Gemma model is not available during this audit.
6. Gemma has not produced a successful draft summary in this environment.
7. Obsidian app launch is not verified in the current sandbox profile.
8. Prompt bus apply needs safer overwrite gates.
9. Router does not route prompt bus context-pack tasks correctly.
10. No merge assistant exists.
11. No course/certificate helper exists beyond routing policy.
12. CC/Codex coordination is still human-mediated.

## Minimum Features For 90 Percent

- Dashboard shows bridge status and separates `visible`, `usable`, and `wired`.
- Prompt bus generates Claude, Codex, and ChatGPT context packs with review-first apply.
- Gemma compresses local context and writes draft summaries reliably.
- Ruflo dependencies are installed in isolation and read-only help/version smoke passes.
- First safe local-worker pilot runs without live actions.
- Merge flow summarizes branch state, active locks, validation, and exact merge commands.
- Obsidian opens reliably or has a documented manual fallback.
- Human approval remains required for all public, money, account, email, job, scraping, posting, and certificate actions.

## 90 Percent Definition

Ghoti should feel materially less manual when it can generate the next Claude prompt, Codex prompt, ChatGPT handoff, Gemma draft summary, dashboard status, and merge-readiness summary from local repo state while preserving human approval gates.

## Honest Verdict

Ghoti is becoming a supervised local coordination stack. It is not yet an orchestrated agent system.
