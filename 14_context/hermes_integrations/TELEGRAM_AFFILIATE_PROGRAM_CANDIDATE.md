# Telegram Affiliate Program - Candidate Only (N+6.10C)

Status: **candidate-only. Not enabled.** `affiliate_program_enabled` defaults false. This
is an idea recorded for later evaluation, not a feature that runs.

## What this is

A note that a Telegram-based affiliate/referral lane is a **possible future** monetization
direction worth exploring later - alongside other candidate lanes (content, video, website,
dropshipping). Recording it as a candidate does not start it.

## Hard rules if it is ever explored

Any future affiliate work must be **legal and TOS-aware** (Telegram's terms, each affiliate
program's terms, and applicable law), and must obey these limits:

- **No spam.**
- **No fake engagement.**
- **No auto-mass messaging.**
- **No deceptive referrals** (no hidden or misrepresented affiliate links).
- **No unauthorized scraping.**
- **No auto-send** - the status bot never initiates outbound messages.
- **Human approval for campaigns** - a person reviews and approves any campaign before it
  runs.

## Decision

Explore later, behind its own feature flag, after a separate audited milestone. Nothing in
this pack sends, posts, recruits, or earns anything.
