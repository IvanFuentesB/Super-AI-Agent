# Sandbox Purpose

This sandbox is a local-first AI and automation workspace for the `ai_sandbox` account.

Primary goals:

- become the clean permanent workspace root for projects, prompts, configs, repos, docs, and experiments
- stay safer and more organized than the main profile
- support later GitHub-backed development
- support later Notion-backed documentation
- grow into a practical assistant and automation environment

Non-goals:

- perfect cloning of the main Windows profile
- broad raw AppData migration
- hype-driven tool sprawl
- opaque automation that is hard to inspect or roll back

Important location decisions:

- Canonical root: `C:\Users\ai_sandbox\Documents\AI_Managed_Only`
- Temporary-only root: `C:\Users\ai_sandbox\AI_Workspace`
- Temporary repo currently outside the canonical root: `C:\Users\ai_sandbox\AI_Workspace\21_repos\core\claw-code`
