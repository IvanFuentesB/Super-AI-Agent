# Main Memory

This is the stable, shared memory for the whole sandbox.

Use this area for:

- what the sandbox is
- current machine and tool state
- current decisions
- workspace conventions
- repo classification summaries
- future stack direction
- local-only vs syncable boundaries

Do not use this area for:

- noisy run logs
- temporary prompts
- per-agent scratch notes
- secrets
- browser/session dumps

Core files:

- `sandbox-purpose.md`
- `current-state.md`
- `workspace-conventions.md`
- `repo-classifications.md`
- `future-stack-direction.md`
- `local-vs-sync-boundaries.md`
