# Future Stack Direction

Current direction:

- local-first where practical
- cloud fallback where it actually helps
- versionable configs, prompts, and scripts
- strong logging and rollback
- small number of tools with clear jobs

First practical model direction:

- Gemma 4 E2B or E4B as the realistic first local candidate
- Ollama as the likely first local runtime

Later direction:

- add editor integration
- add eval and benchmark structure
- add GitHub-backed infra repos
- add curated Notion summaries after integration exists

Explicit non-goal:

- centering the stack on Claw Code before the baseline sandbox is stable
